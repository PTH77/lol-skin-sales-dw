# lol-skin-sales-dw
Hurtownia danych dla League of Legends, koncentrująca się na skinach i mikropłatnościach w grze
Cel projektu
- Stworzenie prostego DW z danymi o sprzedaży skinów w LoL.
- Udostępnienie danych w formie wymiarów (dim_skins, dim_players) i faktów (fact_sales).
- Możliwość łatwej analizy sprzedaży dla data analystów.
